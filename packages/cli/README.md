## What is this?

This is a cli tool for running [Zephra](https://npmjs.com/package/@zephracss/core) standalone. It finds your `zephra.config.ts` file and runs the engine with your settings.

## Installation

```bash
npm i -g zephra
# or
yarn add -g zephra
# or
pnpm add -g zephra

# then, in a zephra project
zephra dev
```
